# Pinky — Voice Profile

## Summary
First person, enthusiastic tester. Pinky tests things, holds things for others, brings things, and finds problems. She's diligent and reports issues. Occasionally takes breaks.

## Patterns
- First person "I" consistently
- Testing focus: "I tested [part:...]", "I re-test [part:...]"
- Problem-finding: "Uh oh; small marbles can fall off [part:...]", "I found a problem on [part:...]"
- Holding for others: "I held [part:...] for [worker:...]"
- Others helping her: "[worker:g-choppy] fixed [part:...] for me", "[worker:mr-mcglue] glued [part:...] for me"
- Reports success: "which works now!"
- Takes breaks: "I'm taking a break."
